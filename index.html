Artificial Intelligence (AI)

Problem Definition

Problem definition refers to the process of clearly identifying and specifying the task that the machine learning model is intended to solve. It outlines the goal of the learning system.

Data

Data used to train the machine learning model. It consists of input-output pairs (features and corresponding labels or targets) that the model uses to learn patterns.

Evalutation

Process of assessing the performance of a trained model on a dataset that it has not seen before (usually the test set or validation set.

Features

Process of identifying and ranking the features (or variables) in a dataset based on how much they contribute to the predictive power of a model.

Modeling

Process of selecting and training an algorithm or model to learn from data and make predictions or decisions

Experimentation

The purpose of experimentation is to identify the best-performing model or approach that addresses the specific problem you're trying to solve

Insurance Products Portfolio

The goal is to gain insights into how each product category is performing, identify growth trends, and assess which segments are driving overall revenue. By analyzing the data year-over-year, the project will identify patterns in revenue growth or decline, helping to inform product strategy, pricing, and marketing efforts.

Company’s Executive Summary YTY Comparison
